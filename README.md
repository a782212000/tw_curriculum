# 目的：

課綱爭議研燒，各式懶人包的位置又太過分散，也無法同時呈現雙方立論的分歧點，
希望可以借用程式上好用的code review 工具，提供統一的比較及討論的地方


# 方式：
    1. 一個課綱一個branch
    2. 先新加不同的branch (year_95, year_98, etc.)
    3. 每個branch底下第一層為科目，第二層為冊，一冊一個檔案，方便多人同時進行
